# Worship Places Images

This directory contains images for the places of worship section.

## Recommended Image Names:

### Christian Churches:
- `st-james-anglican.jpg` - St. James Anglican Church
- `kalk-bay-methodist.jpg` - Kalk Bay Methodist Church  
- `muizenberg-baptist.jpg` - Muizenberg Baptist Church
- `sacred-heart-catholic.jpg` - Sacred Heart Catholic Church

### Islamic:
- `claremont-mosque.jpg` - Claremont Main Road Mosque

### Jewish:
- `rondebosch-synagogue.jpg` - Rondebosch United Hebrew Congregation

### Hindu:
- `cape-town-hindu-temple.jpg` - Cape Town Hindu Temple

### Buddhist:
- `buddhist-retreat.jpg` - Buddhist Retreat Centre

## Image Requirements:
- **Format**: JPEG or WebP preferred
- **Size**: Recommended 800x600px minimum
- **Quality**: High resolution for web
- **Content**: Exterior views of buildings preferred

## How to Add Images:
1. Place your images in this directory with the suggested names
2. Update the WorshipPlaces component to display them
3. Add Image components with proper alt text

## Respect and Guidelines:
- Ensure you have permission to photograph religious buildings
- Be respectful of religious practices when taking photos
- Avoid photographing during services unless permitted
- Consider exterior shots to respect privacy
